\doxysubsection{Detailed Description}
Contains {\itshape all classes and functions} of the \textbackslash{}\+CImg library. 

This namespace is defined to avoid functions and class names collisions that could happen with the inclusion of other C++ header files. Anyway, it should not happen often and you should reasonably start most of your \textbackslash{}\+CImg-\/based programs with 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\#include "{}CImg.h"{}}}
\DoxyCodeLine{\textcolor{keyword}{using namespace }cimg\_library;}

\end{DoxyCode}
 to simplify the declaration of \textbackslash{}\+CImg Library objects afterwards.
