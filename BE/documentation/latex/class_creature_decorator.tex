\doxysubsection{Detailed Description}
Class \mbox{\hyperlink{class_creature_decorator}{Creature\+Decorator}}. 

Classe abstraite de base pour le design pattern decorator. C\textquotesingle{}est cette classe qui indique qu\textquotesingle{}elles sont les méthodes à redefinir.

De plus cette classe délègue dans un premier temps le comportement original à la créature emballée.
