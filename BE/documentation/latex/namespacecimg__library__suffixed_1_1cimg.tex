Contains {\itshape low-\/level} functions and variables of the \textbackslash{}\+CImg Library. 

Most of the functions and variables within this namespace are used by the \textbackslash{}\+CImg library for low-\/level operations. You may use them to access specific const values or environment variables internally used by \textbackslash{}\+CImg. \begin{DoxyWarning}{Warning}

\end{DoxyWarning}
