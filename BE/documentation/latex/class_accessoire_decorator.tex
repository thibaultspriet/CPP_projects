\doxysubsection{Detailed Description}
Class \mbox{\hyperlink{class_accessoire_decorator}{Accessoire\+Decorator}}. 

Cette classe définit les méthodes qui seront réimplémentées par ses classes filles.

Chaque classe fille rédéfini une ou plusieurs méthodes, mais elles ne sont pas obligées de toutes les redéfinir. C\textquotesingle{}est pourquoi \mbox{\hyperlink{class_accessoire_decorator}{Accessoire\+Decorator}}, appelle la méthode de la créature emballée.
