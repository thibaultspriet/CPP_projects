Detailed Description

Class AccessoireDecorator.

Cette classe définit les méthodes qui seront réimplémentées par ses classes filles.

Chaque classe fille rédéfini une ou plusieurs méthodes, mais elles ne sont pas obligées de toutes les redéfinir. C'est pourquoi AccessoireDecorator, appelle la méthode de la créature emballée.
