\doxysubsection{Description détaillée}
Interface qui permet au \char`\"{}client\char`\"{} de créer des objets abstraits de type \mbox{\hyperlink{class_voiture}{Voiture}}. Cette interface délègue la création d\textquotesingle{}un certain type de voiture à ses classes filles. Cette interface met à disposition les méthodes qu\textquotesingle{}on peut appeler sur un objet de type voiture.
